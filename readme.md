
### Bibliotecas Utilizadas
* Backend
    * **express:** Framework para Node.js que simplifica o processo de criar apps e APIs
    * **pg:** Tradutor para o banco PostgreSQL
    * **dotenv:** Leitor do arquivo ```.env```, disponibilizando por ```process.env.DB_USER```, por exemplo
    * **body-parser:** Analisa o corpo das requisições http recebidas, tornando disponivel em req.body
* Frontend
    * **Ejs:** View engine
    * **Axios:** Cuida das requisições
    * **Express-Session:** Guarda o Token JWT do usuário logado
